 # Motor Bike Sales & Revenue Analysis Dashboard

📊 Project Overview

This project showcases an interactive Power BI dashboard that analyzes motor bike sales, revenue trends, and profit margins across different states, cities, and product categories.

The dashboard is designed to help stakeholders understand sales performance, identify high-profit products, and analyze regional sales distribution using interactive visualizations.

📁 Dataset

The dataset contains structured sales information including:

* Customers

* Brands

* Categories

* Order_items

* Orders

* Products

* Staffs

* Stores

Data transformation and modeling were performed using Power Query and DAX inside Power BI.

### Data Transformation Documentation
Overview
This project involves the end-to-end transformation of raw Motor Bike sales data into a high-performance analytical model. The primary goal was to transition from a collection of fragmented, normalized Excel files into an optimized Star Schema within Power BI, enabling accurate business intelligence reporting and time-intelligence analysis.



#### Data Cleaning: 
* Performed data profiling to identify and remove redundant or irrelevant columns that did not contribute to business analysis.

* Standardized column names and data types to ensure consistency across all related tables.

* Resolved data quality issues to ensure accurate relationships between entities.

#### Feature Engineering:

* Implemented custom date hierarchies to support granular time-based reporting (Year, Month, Day).

* Developed core business KPIs using DAX, including Total Sales, Profit Margin %, YTD Sales, and PY Sales to facilitate trend analysis.

* Created calculated columns to bridge gaps in the raw dataset.

#### Data Modeling:

* Designed a robust Star Schema by establishing one-to-many (1:N) relationships between the central fact table (Order Items) and dimension tables (Customers, Products, Stores, and Staffs).

* Ensured proper filter propagation to allow for complex cross-functional reporting across different organizational hierarchies.

📌 Dashboard Features

1️⃣ Sales & Revenue Overview

Key performance indicators showing overall business performance.

KPIs

💰 Total Sales

🛒 Average Purchase Value

🎯 Sales Goal Comparison

2️⃣ Sales Trend Analysis

A line chart using Date Hierarchy to analyze yearly sales performance and trends.

Insights include:

* Sales growth over time

* Year-to-year performance comparison

3️⃣ Sales by State

* A bar chart visualization displaying total revenue generated by different states.

* Helps identify top performing regions and market opportunities.

4️⃣ Profit Margin by Product Category

A category-level analysis showing which bike categories generate the highest profit margins.

Top categories include:

* Mountain Bikes

* Road Bikes

* Cruiser Bicycles

5️⃣ Top 10 Cities by Sales (Map Visualization)

A geographic map displaying top cities contributing to total revenue.

This allows stakeholders to identify regional demand patterns and high-value markets.



 #### Advanced Power BI Features Used

Allows users to navigate from high-level visuals to detailed data views for deeper analysis.

Custom tooltips provide additional insights when hovering over visuals.

🔖 Bookmarks

Bookmarks are used to create interactive navigation and improve dashboard usability.
